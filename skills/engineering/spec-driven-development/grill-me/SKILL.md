# /grill-me — Interview me relentlessly

Interview me relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one. For each question, provide your recommended answer.

Ask the questions one at a time.

If a question can be answered by exploring the codebase, explore the codebase instead.

## When to use

- User says: "grill me", "stress-test this plan", "challenge my design"
- User wants to pressure-test a plan or design before committing
- Starting a new feature or phase and want to resolve decisions before specifying
- Working through a complex design tree with interdependent choices
